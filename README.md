Bank Credit Card Launch Strategy: EDA 

This project involves analyzing bank customer data to identify the optimal group for launching a new credit card.
The analysis includes performing Exploratory Data Analysis (EDA) to uncover target customer segments.

Project Overview
1.This project analyzes customer demographic and financial data to identify patterns in age groups,
income levels, credit scores, and credit card acceptance behavior using exploratory data analysis (EDA).
2.Conducting Exploratory Data Analysis (EDA) to understand customer characteristics and segment them effectively.

Project Structure
1.Bank_Credit_Card_Launch_Strategy_EDA.ipynb - Notebook with EDA to explore customer segments.
2.analysis.png - Dashboard summary of key analysis results.
3.README.md - Project overview and guide.

Exploratory Data Analysis (EDA)
1.Performed data cleaning and exploration to understand customer demographics, income distribution, credit scores, and spending patterns.
2.Used visualizations and summary statistics to identify trends, relationships, and key differences across age groups and customer segments.

Technologies Used
1.Python: Data processing and analysis
2.Pandas,NumPy: Data manipulation
3.Matplotlib, Seaborn: Data visualization

final result:
1.Credit card acceptance and financial behavior vary across age groups, indicating the need for age-based customer targeting.

The analysis provides basic insights into customer behavior that can support better decision-making.
